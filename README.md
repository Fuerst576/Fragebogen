# Fragebogen – Interactive Quiz Platform

Eine web-basierte Quiz-Anwendung mit Benutzerauthentifizierung und MySQL-Datenbankintegration. Nutzer können sich registrieren, ein interaktives Quiz absolvieren und ihre Ergebnisse speichern.

## ✨ Features

- **Benutzerauthentifizierung**: Registrierung und Login-System mit Passwortschutz
- **Interaktives Quiz**: Multiple-Choice- und Bildauswahl-Fragen mit verschiedenen Fragetypen
- **Datenspeicherung**: Automatische Speicherung von Benutzerantworten in der MySQL-Datenbank
- **Ergebnisverfolgung**: Punktevergabe und Ergebnisauswertung mit Zugriff auf Responses
- **Responsive UI**: Benutzerfreundliches Interface mit CSS-Styling und dynamischem Hintergrund

## 🛠️ Technologien & Tools

- **Backend**: PHP (OOP mit Klassen: `Auth`, `Quiz`)
- **Datenbank**: MySQL / phpMyAdmin
- **Frontend**: HTML5, CSS3
- **Session Management**: PHP Sessions für Authentifizierung
- **Architektur**: MVC-ähnliche Struktur mit Klassen-basierter Logik
